# 🧮 Simple Calculator (Python Tkinter)

A basic GUI-based calculator application built using **Python** and **Tkinter**.  
This project allows users to perform simple arithmetic operations with an easy-to-use interface.

---

## 🚀 Features

- ✅ User-friendly graphical interface  
- ✅ Supports basic operations:
  - **Addition**
  - **Subtraction**
  - **Multiplication**
  - **Division**
- ✅ Real-time result display  
- ✅ Error handling for:
  - Invalid inputs
  - Division by zero  

---

## 🛠️ Technologies Used

- **Python**
- **Tkinter (GUI Library)**

---

## 📷 Application Workflow

1. Enter the first number  
2. Enter the second number  
3. Select an operation from the dropdown  
4. Click the **Calculate** button  
5. View the result instantly  

---

## ⚠️ Error Handling

- Shows alert if input is not a number  
- Prevents division by zero with an error message
