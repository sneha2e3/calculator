# calculator
This is a basic GUI-based calculator application developed using Python and the Tkinter library. The application allows users to perform simple arithmetic operations such as addition, subtraction, multiplication, and division through an interactive graphical interface.

#FEATURES:
User-friendly graphical interface
Supports basic operations:
Addition
Subtraction
Multiplication
Division
Input validation with error handling
Displays results instantly

#Technologies Used:
Python
Tkinter (for GUI development)

#How It Works:
Enter two numbers in the input fields
Select the desired operation from the dropdown menu
Click the Calculate button
The result will be displayed on the screen
